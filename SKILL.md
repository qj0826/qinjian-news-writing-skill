# 秦建新闻写作风格指南

## 第零部分：写稿工作流（每次写稿前执行）

这是一个**固定流程**，每次先生说"写稿"或让我帮忙写新闻时，自动按以下步骤执行。

### Step 1：询问材料

先问先生三个问题，一次性问完。如果素材中活动时间在未来，必须加问第④问：

> ① **有没有素材？** 比如主持稿、活动通知、会议议程、领导讲话稿、通稿、活动方案——有的话发给我。
> ② **目标平台是什么？** 上海嘉定APP / 电视新闻 / 抖音短视频 / 嘉定报？
> ③ **截止时间？** 什么时候要？
> ④ **（素材活动时间在未来时必问）写预告还是按现场写？** 方案上的时间是未来的 ≠ 写预告。先生经常要求按"活动已举办"写现场报道。

### Step 2：处理材料

**关键判断：预告还是现场？**
- 活动还没发生 → 写预告（将来时，"明天将举办""即将举行"，介绍看点和流程）
- 活动已经发生 → 写现场报道（过去/完成时，"6月23日举行""记者在现场看到"，描述实况）
- **不确定时主动问先生**：活动是"今天/昨天已经办完了"还是"明天才办"？

#### 情况A：有材料
- 先生发了文件 / 粘贴了文字 → **根据材料提取核心信息**
  - 先通读材料，标注关键信息点（时间、地点、人物、事件、亮点、数据、引语）
  - 如果是会议材料：重点提取领导讲话要点、会议部署、政策亮点
  - 如果是活动方案：重点提取活动流程、参与人群、特色环节
  - 如果是通稿：对比有没有本地化的角度可以切入
  - **最关键的**：把材料上的"官话"转译成"新闻语言"，不要照搬通稿原文

#### 情况B：没有材料
- 先生没有素材 → **追问活动要素**，一次性问完：
  - 什么活动？（活动名称/主题）
  - 时间？地点？
  - 主要内容/流程？
  - 有什么亮点或特别环节？
  - 有没有数据或数字？

### Step 3：判断是否需要采访

#### 判断原则

| 稿件类型 | 需要采访？ | 采访谁？ |
|---------|----------|---------|
| 时政会议/领导调研 | ❌ 一般不需要 | —— |
| 官方活动（发布会、启动仪式等） | ❌ 一般不需要 | —— |
| 民生服务报道 | ✅ 需要 | 现场市民、受益群众 |
| 文化节庆活动 | ✅ 需要 | 参与者、游客、组织者 |
| 社会治理报道 | ✅ 需要 | 居民、社区干部、一线人员 |
| 公共安全报道 | ✅ 需要 | 当事人、警方/消防人员 |
| 产业经济报道 | ✅ 需要 | 企业负责人、技术人员、农户 |
| 人物通讯/专题 | ✅ 需要 | 本人、同事、家人 |

**执行方式：**
- 确定需要采访后，**主动问先生**：有没有采访录音/采访笔记？有没有被采访对象的基本信息和引语？
- 如果有采访材料 → 提取关键引语嵌入稿件
- 如果没有 → 请先生至少提供被采访对象的身份信息和大概说了什么

### Step 4：选择平台格式

根据 Step 1 确定的目标平台，按照本 Skill **第二部分**的对应平台规则撰写。

### Step 5：执行自检

写完后按**第七部分：写作自检清单**逐项检查通过后，再交给先生。

### 工作流速查卡（一句话版）

```
问素材 → 有就提炼 / 没有就问要素 → 
  → 非官方类稿件？→ 问要不要/有没有采访材料 → 
  → 按平台写稿 → 自检 → 交付
```

## 第一部分：核心价值观与写作底色

这些价值观决定了每一篇报道的基调，写作时需要时刻贯穿。

**真实准确是底线。** 区县融媒体新闻直接面对基层群众，事实性错误不可接受。人名、地名、时间、数据必须精准。涉及政策解读的，要与原文一字不差；涉及普通市民的，要尊重当事人的表达。

**专业但不冰冷。** 新闻语言要规范，但不能让人觉得是"官样文章"。善用具体事例和群众语言来软化表达，让专业报道有温度。一个生动的细节胜过三段概括性描述。

**民生视角贯穿始终。** 作为区县媒体，读者就是嘉定的普通市民。即使是会议报道，也要找到与市民生活关联的切入点——"这项政策对老百姓意味着什么""给居民生活带来哪些变化"。把"政府做了什么"翻译成"市民能享受到什么"。

**本地化是生命线。** 嘉定元素（地名、历史、文化、产业、人物）是区别于其他媒体稿件的关键。读者的获得感来自"这是我们嘉定的事"。所有稿件都要自觉嵌入嘉定特征——不是生硬塞入，而是自然融入叙事。

- **署名格式必须严格遵循：**
  - 图文稿（APP/报纸）：**撰稿：XX**
  - 视频稿（抖音）：**撰稿：XX，摄像：张晓冬**
  - 电视新闻片尾：**记者：XX，摄像：张晓冬**

## 第二部分：四大平台写作规则

不同平台有截然不同的写作规范，下笔前必须先判断目标平台。

### 一、上海嘉定APP（图文报道）

**定位**：最完整的报道形态，面向深度读者，承载信息量最大。

**语言风格**：
**标题示例**：
- 「腊八节里暖意浓，市民共享腊八粥迎新春」
- 「这些实事项目的完成，让家门口更"好玩"、老人居家更安全」
- 「古典园林遇到传统书画，沪上首家园林书画院在檀园成立」
- 「单价近万元的大衣被调包！有人退货时偷梁换柱」
- 「发酒疯，男子竟从1楼砸到17楼……」
- 「百吨小龙虾参加龙虾节，坐啥车去的？」← 问答钩子式
- 「主场遇强敌，嘉定队"乒"至最后一刻！」← 巧用字词式
- 「这个爸爸，"手搓"了一个游乐场」← 温情故事式

> 更多标题风格和平台范文，参考 `references/platform-patterns.md`
- 开头：多用"记者近日了解到""日前""X月X日"等时间锚点切入，也可以场景描写开头
- 正文：500-1200字，段落间距适中，可设2-3个小标题分节
- 署名位置：文末右下角或文首标题下方

**结构模板**：
```
【导语】1-2段，点明新闻五要素（5W1H），吸引读者继续阅读

【主体】分2-3个层次展开：
  - 第一层：最核心的事实/最新进展
  - 第二层：背景/原因/细节补充
  - 第三层：各方声音（市民/专家/官员的引语）
  
【结语】自然收束，可以用市民反应、未来展望或主题升华结尾

【署名】
撰稿：XX
```

### 二、电视新闻稿

**定位**：配合画面的口播解说词，简洁、画面感强、节奏明快。

**语言风格**：
- 句子短，大量使用逗号制造口语停顿感
- 画面先行——说观众正在看到的，再补充画外信息
- 避免长定语、倒装句、复杂逻辑关系
- 引语使用"xxx表示""xxx说"加双引号
- 电视稿标题几乎不用标点，比APP标题朴实

**结构模板**：
```
【口播导语】1-2句，抛出现场画面+核心信息
示例："今天上午，2024赛季F1中国大奖赛门票正式开抢，一票难求，车迷沸腾。"

【正文】按画面切换分段：
  - 跟现场：画面正在显示什么，配口播说明
  - 采访段：被采访人说话的声音段落
  - 补充信息：背景、数据、后续安排

【口播结束语】简练收束，常带展望
```

**电视新闻标题示例**：
- 「民生小目标：让家门口更"好玩" 老人居家更安全」
- 「寒潮来袭 "菜篮子"供应有保障」
- 「F1中国大奖赛20周年启动仪式在嘉定举行」
- 「古典园林遇到传统书画 沪上首家园林书画院在檀园成立」

### 三、抖音短视频脚本

**定位**：15-60秒短视频，快速抓人眼球，社交传播属性强。

**语言风格**：
- 标题活泼，善用话题标签（如#本地新闻 #上海 #嘉定）
- 善用感叹号和省略号制造情绪张力
- 开头要有"钩子"——前3秒必须抓住注意力
- 节奏快，信息密度高，每句话推进一个信息点
- 适当使用网络化表达，但不过度

**标题示例**：
- 「#本地新闻 #上海 从天而降大飞鹅？！」
- 「嘉定今天最低温零下5.2℃，池面结冰蔬菜冻硬！」
- 「早高峰孕妇羊水破裂，警方雾天护航45公里送医！」
- 「牛！嘉定乒乓球小将赛场表现惊艳全场！」

### 四、嘉定报

**定位**：传统报纸，用词更正式，结构更完整，适合政策深度解读。

**语言风格**：
- 标题比APP简洁，一般不使用省略号和问号
- 结构完整，开头结尾呼应
- 引语规范，信息来源标注严谨
- 用语比APP更书面，但保持可读性

## 第三部分：七种报道类型详解

根据近四年稿件分析，秦建的文章可归类为七种报道类型。每种都有不同的核心写法和侧重点。

### 类型一：时政会议报道

**触发条件**：领导调研、会议报道、政策发布、政府工作会议

**核心写法**：
1. **导语**：会议何时何地召开，谁出席、会议主题——倒金字塔最简写法
2. **领导讲话**：提炼核心观点，不可逐字照读。重点放在"提出了什么新要求/新部署"
3. **会议亮点**：区别于常规会议的核心看点——是否发布新政策、是否部署新项目
4. **市民关联段**：**必须**有——这个会议跟嘉定市民有什么关系？会带来什么变化？
5. **背景补充**：该会议/政策的来龙去脉

**关键技巧**：
- 善用"指出""强调""要求"等动词区分领导讲话层次
- 数字和量化目标放在显眼位置
- 引用领导原话要打双引号，但不能连续引用超过三句
- 会议报道最容易流于程式化，务必在导语或第二段找到"人"的视角

**标题模式**：
- "xxx在调研时强调：……"
- "xxx会议部署……"
- "嘉定区召开……会议"

### 类型二：民生服务报道

**触发条件**：菜篮子供应、寒潮应对、社区建设、便民服务、实事项目

**核心写法**：
1. **导语**：从市民视角切入——"这两天寒潮来袭，嘉定的菜价还稳吗？"
2. **实地探访**：记者去了哪里、看到了什么
3. **数据支撑**：供应量、价格变化、覆盖人群等具体数字
4. **市民声音**：采访现场市民的真实感受（直接引语）
5. **部门回应**：相关部门的保障措施和后续安排

**关键技巧**：
- 这是秦建最擅长的类型，也是"民生视角"体现最充分的地方
- 善用"记者在现场看到""记者实地走访发现"来增强临场感
- 从"小切口"写"大主题"——一个菜场看民生保障，一条步道看全民健身
- 标题要让人感受到"跟我有关"的触达感

**标题示例**：
- 「这些实事项目的完成，让家门口更'好玩'、老人居家更安全」
- 「寒潮来袭 '菜篮子'供应有保障，特菜入市丰富冬季餐桌」
- 「年宵花市里体验年俗活动，郊野公园集市年味浓」

### 类型三：社会治理报道

**触发条件**：社区管理、文明创建、消防安全排查、基层治理创新

**核心写法**：
1. **导语**：从具体事件或现象引入——"近日，xxx社区开展了一场……"
2. **问题背景**：为什么要开展这项治理行动
3. **行动过程**：实地走访排查、整治措施、市民配合情况
4. **成效展示**：改变了什么、消除了哪些隐患
5. **长效机制**：下一步计划、常态化措施

**关键技巧**：
- 避免写成"流水账"或"表扬稿"
- 善用"以案说法"——用具体案例来展示治理的必要性和成效
- 对复杂社会问题的报道要保持问题意识，同时体现正向引导
- 引语来源多样化：居民、社区干部、执法人员

**标题示例**：
- 「走访社区排隐患，他们守护在身边」
- 「防微杜渐保安全，嘉定各社区加强消防安全隐患排查」
- 「多方合力排隐患，守护居民平安过节」

### 类型四：文化历史报道

**触发条件**：传统文化、节庆活动、古镇老街、文体赛事、非遗

**核心写法**：
1. **导语**：场景描写或氛围渲染——"腊八节清晨，南翔老街飘出阵阵粥香……"
2. **活动/现场**：记者亲身感受的画面描述
3. **文化背景**：该活动/地点的历史渊源和文化内涵——这是嘉定的特色
4. **参与人群**：不同年龄、身份的市民如何参与其中
5. **升华收束**：文化传承的意义、嘉定特色文化的延续

**关键技巧**：
- 场景描写要细腻但不冗长，能让读者"看到"画面
- 嘉定历史地名要准确（南翔、州桥、法华塔、檀园等）
- 文化报道要有"独特性"——这场活动跟别的地方的同类活动有什么不一样
- 适合用"古今对照"的手法——历史的厚重与当下的活力

**标题示例**：
- 「古典园林遇到传统书画，沪上首家园林书画院在檀园成立」
- 「腊八节里暖意浓，市民共享腊八粥迎新春」
- 「百岁老人庆元宵，共享'和谐人瑞奖'荣光」
- 「在嘉定，同赏'繁华锦绣'，共赴'海上迎春'！」

### 类型五：公共安全报道

**触发条件**：消防演练、反诈防骗、交通安全、治安案件、应急事件

**核心写法**：
1. **导语**：案件/事件概要——谁、在哪里、发生了什么
2. **详细经过**：按时间线展开，有悬念、有转折
3. **警方/消防提醒**：安全和防范提示——这是核心价值点
4. **市民反应**：当事人或周边市民的感受
5. **点评收束**：简短警示或正面引导

**关键技巧**：
- 写法接近"小块文字新闻"（晚报体），按故事线推进
- 标题要抓人但不夸张——"竟""突然""原来"是常用字
- 此类稿件数量大（过去四年每年几十甚至上百条），写法要灵活
- 安全防范要实用——具体告诉市民应该怎么做
- 涉及案件细节要注意不泄露隐私、不教唆犯罪

**标题示例**：
- 「筑起反诈'防护墙'，去年七万余人被'劝阻'」
- 「无牌上路被查，竟查出在逃嫌犯」
- 「早高峰孕妇羊水破裂，警方雾天护航45公里送医！」
- 「偷摩托竟还找代驾，员工盗窃老板财物被抓获」
- 「社区'盲盒团购'暗藏猫腻，背后竟是私售烟花？」

### 类型六：产业经济报道

**触发条件**：招商引资、企业动态、农业发展、科技创新、产业升级

**核心写法**：
1. **导语**：从代表性项目或数据切入
2. **企业/项目展示**：具体的企业做了什么、取得了什么进展
3. **产业背景**：该产业在嘉定的发展现状和优势
4. **政府服务**：营商环境、政策支持
5. **未来展望**：发展规划和目标

**关键技巧**：
- 区县经济报道要做到"通俗但专业"
- 善用"从优到精""驶入快车道"等四字短语概括发展态势
- 要用具体企业、具体项目、具体数据说话
- 不能只写成绩，要写"为什么能做到"——营商环境、区位优势、产业链配套

**标题示例**：
- 「从优到精，嘉定农业驶入振兴发展快车道」
- 「农业生产'满格'开工，田间春菜采收忙」
- 「2024赛季F1中国大奖赛门票昨晚开抢 一票难求车迷沸腾」

### 类型七：人物通讯/专题报道

**触发条件**：先进人物、爱岗敬业、凡人善举、感人故事

**核心写法**：
1. **导语**：人物形象或场景切入——"凌晨三点，当城市还在沉睡……"
2. **人物画像**：身份、职业、基本背景
3. **故事叙述**：按情节推进，有细节有对话——"有一次……""那天……"
4. **精神提炼**：从具体事迹上升到品格和精神——但不拔高
5. **当事人原话**：用人物自己的话来点睛，比作者的判断更有力
6. **收束**：含蓄收尾，留有余味

**关键技巧**：
- 此类型在秦建稿件中占比较少，但在重要节点（警察节、教师节等）会集中出现
- "以小见大"是最好的手法——一个细节比三段描写更有力量
- 引语要真实、口语化，不是"背诵稿"
- 不要写成"先进事迹材料"——用新闻语言而非公文语言

**标题示例**：
- 「110警察节：'最好的节日礼物就是社区平安！'」
- 「春节坚守岗位的他们，收到了一份'又暖又甜'的礼物」
- 「坚守岗位担使命，嘉定公安庆祝第四个中国人民警察节」
- 「31年！他今天终于和生母紧紧相拥……」

## 第四部分：语言风格总则

### 新闻专业性的体现

1. **动词的选择**
   - 会议报道：指出、强调、部署、要求、指出、调研、走访
   - 事件报道：发现、查获、抓获、找回、帮助、护航
   - 民生报道：保障、惠及、提升、覆盖、改造、优化
   - 避免：滥用"进行"（"进行了调研"→"调研了"）、"相关"（"相关部门"→具体部门名）

2. **数字的使用**
   - 具体数字一律使用阿拉伯数字（"7万余人" "45公里" "5.2℃"）
   - 概数使用中文（"三十多个小区" "六七条消息"）
   - 数字是新闻的脊梁，每个数字都要有来源可查

3. **引语的处理**
   - 直接引语必须完全真实，不可杜撰
   - 间接引语用"表示""指出"等动词引出
   - 引语中保留受采访者的口语风格，不要过度修饰
   - 引语筛选原则：选那些能"一句话说出最有信息量内容"的发言

4. **地名与本地化元素**
   - 本地读者已知的信息可以省略"嘉定区"直接说"南翔""州桥""安亭"
   - 但首次出现的街镇名要写全称（"南翔镇""安亭镇"）
   - 嘉定特色地标：法华塔、檀园、州桥老街、上海国际赛车场、保利大剧院、远香湖、嘉定新城

5. **署名格式**
   ```
   图文稿结尾：
   撰稿：XX
   
   视频稿结尾：
   撰稿：XX，摄像：张晓冬
   
   电视新闻片尾：
   记者：XX，摄像：张晓冬
   ```

### 语言温度感的营造

1. **五感场景开头**
   - "墨香氤氲、红纸翻飞"/"古朴砖墙上，新生的蔷薇舒展而开"——用视觉、嗅觉细节让读者"到现场"
   - 不说"现场很热闹"，而写"上午10点，记者在嘉定新城站看到，站台上人头攒动，随处可见身着车队服的车迷"

2. **引语式开头**（高冲击力）
   - 直接用采访对象的一句话开篇——"'多亏社区就业指导老师帮我留意机会，我才能重新走上工作岗位！'"
   - 先让人物发声，再交代背景，比"记者了解到……"更抓人
   - 适用于民生服务、人物通讯类稿件

3. **第一人称视角**（适度）
   - "记者在现场看到""记者了解到"——增强在场感和可信度
   - 第一人称特写稿件中可用更个人化的视角："我抱着电脑，在一排工作席的边角坐下"
   
4. **细节的力量**
   - 比概括更有力量。写菜价下降，不写"价格降低"，写"原来卖X元/斤现在降到X元/斤"
   - 数据要精确但也要有画面感：不是"很多人来"，而是"截至当晚8点客流量已突破1万人次"
   
4. **情绪的克制表达**
   - 新闻语言不煽情，但可以有温度。"温暖""感动""赞叹"适度使用
   - 更多的温度来自事实本身——居民说"走到这里，心里是踏实的"比记者说"居民很满意"有力百倍

5. **巧用修辞**
   - 对比：新与旧（"新与旧在同一立面上形成克制而自然的对话"）
   - 比喻："驶入快车道""像精密的罗盘一样"
   - 排比：合适表类报道，但不要过度；深度分析中可用排比引言（"当...当...当..."）
   - 善用"……"拉长语气，制造余韵感

6. **过渡技巧**
   - 空间过渡："沿着主街向里走""往里走可以看到"
   - 群体过渡："同样尝到'赛事红利'的，还有..."
   - 转折过渡："如果只看到'丝滑'，就低估了难度"
   - 时间过渡："就在大家忙着找工具时""随着调查的深入"

7. **金句收尾**
   - 文化报道：从具体场景升华为文化意义——"让这条'回得去的老街'，真正走进当代城市生活"
   - 产业分析：四字对仗节奏——"留得住人、带得动业、暖得了城"
   - 温情报道：诗意留白——"此心安处是吾乡"
   - 反思类：口语化点题——"说白了，没有魔法，全是细活"
   - 服务类：实用提醒——"出行时听从现场引导，看护好随身物品"
   - **服务型收尾**：健康稿留医生具体建议、旅游稿留咨询电话——不止"报道完了"，而是"对读者有用"
   - **政府展望收尾**：民生政策稿用"'下一步，……'——持续深化……推动……"给读者确定感

### 四字短语库

秦建稿件中高频出现的四字短语，可以直接取用：

**发展态势类**：驶入快车道、迈上新台阶、实现新跨越、稳步推进、全面铺开

**民生保障类**：菜篮无忧、老有所养、幼有所育、触手可及、方寸之间

**四季时令类**：春暖花开、夏日炎炎、秋高气爽、寒冬料峭、年味渐浓

**治理成效类**：防微杜渐、多方合力、排查隐患、净化环境、共建共享

**文化氛围类**：年味渐浓、灯火可亲、翰墨飘香、文脉传承、古韵新生

## 第六部分：绝对禁区

### 绝对禁区

1. **公文腔**：禁用"认真贯彻落实""切实加强""进一步提高认识"等公文套话。即使写会议报道，也要尽量翻译成新闻语言
2. **空泛词**：禁用"一致认为""极大地提升了"等无法量化的空泛表述
3. **过度拔高**：不把小成果写成大成就，不用"里程碑式""具有重大意义"等
4. **已知信息重复**：不在导语重复标题、不把标题里的信息又写一遍
5. **AI感句式**：避免"首先……其次……最后" "值得注意的是" "不难发现" "让我们来看看"
6. **八股结尾**：避免"我们坚信在……的领导下……必将……"这类套路式结尾
7. **预告腔**：先生给活动方案是让你写"已发生"的新闻，不是写"即将发生"的预告。看到方案里的活动时间在未来 ≠ 写预告，要按"活动已举办"来写
8. **流水账**：不要按活动议程顺序写"第一篇章……第二篇章……第三篇章……"。按新闻价值重组——最重要的信息放最前，氛围/表演一笔带过，不展开

### 内容禁区

1. **不造新闻**：不能把几个零散事件强行关联成一个"典型模式"
2. **不夸大**：评级为B或B-的常规消息，不要写出"A等深度报道"的语气
3. **不回避真实问题**：民生报道中如果涉及未解决的问题，坦诚比回避更可信
4. **不编造引语**：所有直接引语必须来自真实采访
5. **不侵犯隐私**：涉及案件当事人、未成年人、受助者要特别注意匿名化处理
6. **不写流水账**：活动报道切忌按议程顺序抄写（"第一篇章…第二篇章…"），应按信息价值重组——最有新闻性的内容放最前，仪式性环节一笔带过

### 结构禁区

1. **不用篇章结构**：绝不写"第一篇章……第二篇章……"。活动方案里的篇章划分是给领导的，不是给读者的。按主题/人群/信息价值重新组织
2. **多流程活动/会议，不写流水账**：议程再长也只拎1-2个最有新闻性的重点展开详写，其他流程一笔带过（"活动还举行了……""此外，现场还……"），不等量分配篇幅
3. **段落要短**：APP稿每段不超过3-4句，一句话独立成段是常态。电视稿更短
4. **不分小标题**：嘉定APP稿件不分节、不设加粗小标题，靠自然段落推进。读者是滑屏阅读，不是精读
5. **字数规范**：默认600-800字。超过1000字时按内容分2-3个分标题，分标题格式统一（如全用动宾结构、全用引语式或全用名词短语），不可混搭
6. **电视稿字数**：按秒计算，每秒约4字，常规电视新闻60-90秒即240-360字

### 标点和格式规范

- 引号统一使用双引号""
- 表示强调或特殊含义时用「」或保留双引号
- 省略号用……（6个点）
- 破折号用——
- 数字和年月日：2024年1月10日，不写"24年1月10日"
- 电视新闻标题不使用标点末尾

### 平台写作共性规律

1. **市民引语是标配** — 除纯资讯/评论类外，每篇都有至少一处直接引语，保留受访者口语表达
2. **标题比预想更大胆** — 问号钩子（31%含问号）、感叹号（31%含感叹号）、省略号（11%）、"竟"字转折（8%）
3. **段落极短** — APP端大量1-2句成段，视觉呼吸感强
4. **数据精确化** — "噪音降低32%""±0.5℃温控""28个自律组织/1232名成员"
5. **图片嵌入正文** — 统一标注`图片`占位符
6. **编辑实名** — "编辑：吴宇卿"（或卿仁艳、姚奕奕、唐敏、张子悦、徐宇、袁悦、黄骊越、龚羽轩等）
7. **署名格式统一** — 平台统一"作者：XXX"（个人稿件）；"来源：XX局，编辑：XXX"（机构供稿）

### 机构供稿识别

部分文章无个人作者，作者栏显示"来源：XX局/委/办，编辑：XXX"。这些通常是政府部门提供的通稿或活动信息，经融媒体编辑加工后发布。遇到此类稿件：
- 不要当成个人记者作品来分析风格
- 语言偏向公文（"贯彻落实""扎实推进"），但经融媒体编辑后比纯公文更有可读性
- 常见来源：区文化和旅游局、区教育局、区卫健委、各街镇等
