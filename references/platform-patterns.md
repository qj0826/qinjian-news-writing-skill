# 上海嘉定APP平台稿件模式参考

> 数据来源：实际阅读 jiading.gov.cn/xinwen/ 平台 2026年6月稿件（92篇，涵盖24位记者）
> 更新日期：2026-06-23（v1.1.0）

## 平台习惯

### 署名格式
- **个人稿件**：平台统一格式 **"作者：XXX"** + 换行 + **"编辑：XXX"**
- **机构供稿**：**"来源：XX局/委/办，编辑：XXX"**（无个人作者）
- 常见编辑（已确认12位）：吴宇卿、卿仁艳、姚奕奕、唐敏、张子悦、徐宇、袁悦、黄骊越、龚羽轩、沈天姣、杨玉洁、陆王淑怡、倪丹丹
- 文章来源标注：**"来源：上海嘉定"**
- **重要**：部分文章的作者/编辑信息在正文末尾（而非顶部信息栏），特别是机构供稿和系列稿

### 文章分类
- **嘉定头条**：重大时政（签约、会议）、领导调研
- **嘉定动态 > 民生新闻 (msxw)**：社区、活动、体育、文化、人物
- **嘉定动态 > 社会新闻 (shxw)**：社会治理、科普、产业、暖心故事
- **嘉定动态 > 经济新闻 (jjxw1)**：招商引资、企业发展
- **嘉定动态 > 时政新闻 (szxw)**：政法、党建
- **嘉定人物**：人物专题
- **长三角动态**：区域合作

### 文章字数实际分布（基于92篇样本）
- 最短：340字（手工教程类）
- 最长：2320字（深度专题类）
- 中位数：~900字
- 常规报道：600-1000字
- 评论/时评（"小嘉说"）：1500-2000字
- 深度专题（"何以嘉定"）：2000+字
- 手工教程（"我嘉手工课"）：300-500字

## 标题风格（实际案例）

### 问答钩子式
- "百吨小龙虾参加龙虾节，坐啥车去的？"
- "关于'门责制'，你了解多少？"
- "能饮一杯无？"

### 巧用字词式
- 主场遇强敌，嘉定队**"乒"**至最后一刻！（乒=乒乓球+拼）
- 汛期保**"畅"**，嘉定企业"技术驰援"湖南（畅=畅通+长沙）
- **"手搓"**了一个游乐场（网络语言）

### 感叹式
- "请你吃面！1000份面条，免费送！"
- "来挑壁纸！这里的夏日美景上线！"
- "牛！嘉定乒乓球小将赛场表现惊艳全场！"
- "QQ弹弹~这是啥？"

### 悬念式
- "早高峰孕妇羊水破裂，警方雾天护航45公里送医！"
- "嘉定街头惊现孔雀，围观市民：飞起来像凤凰"
- "突发意外！双肺动脉严重堵塞，嘉定医生的操作救了她一命……"
- "惊险一幕！发现时已半身没入水中，幸好有她……"

### 温情式
- "这个爸爸，'手搓'了一个游乐场"
- "31年！他今天终于和生母紧紧相拥……"
- "春节坚守岗位的他们，收到了一份'又暖又甜'的礼物"

### 数据式（新发现）
- "入住率翻了一番！中外球迷在'嘉'共赴世界杯之约"
- "F1首日客流同比暴涨近50%"

### 系列标识式（新发现）
- 用"｜"或"丨"分隔系列名和标题：如"汇龙潭｜端午粽香""小嘉说丨前路漫漫""何以嘉定｜他用20年热爱…"

## 正文结构模式

### 模式一：活动报道（戴懿昕风格）
```
导语段：一句话概括事件（时间+地点+活动名+核心亮点）
现场段：场景描写（"上午九点，广场上已是人头攒动"）
活动段：具体环节描述，每个环节2-3句
市民段：至少1-2处市民直接引语
数据段：活动成效、覆盖人数等量化信息
官方段：领导/相关负责人总结发言
收尾段：未来展望或一句话收束
```

### 模式二：产业/科技报道（毛佳凯风格）
```
导语段：趣味钩子（"热闹的人间烟火背后..."）
企业段：企业做了什么、技术是什么
技术段：具体参数、对比数据（噪音降低32%、±0.5℃控温）
引语段：企业负责人直接引语
展望段：未来规划
```

### 模式三：体育报道（戴懿昕风格）
```
导语段：比赛结果一句话
背景段：赛事背景和意义
过程段：关键局/关键节点描述
数据段：比分表（独立区块）
引语段：观众/参与者引语
展望段：下场比赛预告
```

### 模式四：人物/人文（印沁沁风格）
```
导语段：人物或场景切入，温暖直接
故事段：具体事迹，有细节有对话
升华段：精神层面的提炼，但不拔高
收尾：含蓄隽永
```

### 模式五：公共安全报道（席凌燕/张子悦风格，新发现）
```
导语段：时间+地点+事件概要（警情式）
经过段：救援/事件详细经过，按时间线推进
人物段：消防员/民警/当事人原话还原现场
提醒段：安全提示或防范建议
收尾：简短收束，常有正面引导
```

### 模式六：散文/随笔（"汇龙潭"系列，姚鸣风格，新发现）
```
导语段：年代/场景引入（"上世纪六十年代""小时候"）
叙事段：个人回忆，细腻生活细节
情感段：含蓄但有温度的情感表达
收尾：余韵悠长的结尾，常带感叹或反思
```

### 模式七：评论/观点（"汇龙谈"/"小嘉说"系列，新发现）
```
导语段：问题揭示或现象描述
论述段：以案说法，多角度分析
观点段：核心观点亮出
收尾：建议/呼吁/展望
```

### 模式八：深度专题（"何以嘉定"系列，新发现）
```
编者按：1段引出主题和意义
人物叙事线：人物历程、细节、访谈
产业观察线：行业发展、数据、趋势
双线交织：人物故事+产业分析交替推进
收尾：升华主题，留有余味
```

### 模式九：机构供稿（新发现）
```
格式特点：
- 无"作者：XXX"，改为"来源：XX局/委/办，编辑：XXX"
- 语言偏向公文但经融媒体编辑加工
- 常见于政策发布、对外交流、统计信息类稿件
- 不纳入个人记者风格分析
```

## 共同特征

1. **段落极短**：APP端每段平均2-4句，一句话独立成段常见
2. **图片标注**：文中配图标注"图片"字样
3. **直接引语标配**：除评论/资讯类外，每篇必有市民或参与者的口语化引语
4. **数据意识强**：能用数字就不用形容词（"28个自律组织"比"多个组织"好）
5. **口语但不随意**：标题口语化，正文保持新闻语言规范性
6. **无"小标题"**：文章不分节、不设小标题，靠自然段落推进
7. **来源统一**：全部标注"来源：上海嘉定"
8. **系列标识**：固定系列用"｜"或"丨"分隔系列名和标题
9. **署名位置不固定**：个人稿件顶部"作者：XXX"，机构稿/系列稿可能在正文末尾
