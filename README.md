# JCMCwriting.skill

**JCMC 新闻写作风格 Skill**

用于 测试AI辅助撰写新闻稿件。

---

## 这是什么

这是上海嘉定融媒体中心记者写作风格文件（Skill），基于：

- **838 篇**实际稿件统计分析
- **4 年**（2021–2025）写作数据挖掘
- **7 种**报道类型精炼
- **11 位**同平台记者横向参考
- **4 大平台**切换规则（上海嘉定APP / 电视新闻 / 抖音 / 嘉定报）

加载此 Skill 后，AI Agent 会按照JCMC的新闻语言、结构习惯和编辑规范撰写稿件。

---

## 包含内容

| 板块 | 说明 |
|:-----|:-----|
| 写稿工作流 | 5 步固定流程：问素材 → 提炼/追问 → 采访判断 → 按平台写 → 自检交付 |
| 核心价值观 | 真实准确、专业不冰冷、民生视角、本地化 |
| 四大平台规则 | APP 图文、电视口播、抖音脚本、报纸深度稿 |
| 七种报道类型 | 时政会议 / 民生服务 / 社会治理 / 文化历史 / 公共安全 / 产业经济 / 人物通讯 |
| 语言技法 | 五感开头、引语开头、过渡技巧、金句收尾、四字短语库 |
| 写作模板 | 6 套可直接套用的结构模板 |
| 绝对禁区 | 公文腔、空泛词、AI 感句式等 11 条红线 |
| 自检清单 | 写后逐项核查 |
| 同侪参考 | 11 位记者写法分析 + 平台共性规律 + 838 篇统计数据 |

---

## 如何使用

1. 将 `JCMCwriting.skill` 放入 AI Agent 的 skills 目录
2. 写稿时加载此 Skill
3. 按工作流自动执行：先会问素材/平台/截止时间，再按对应规则出稿
