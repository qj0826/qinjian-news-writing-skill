# 秦建新闻写作风格指南 (qinjian-news-writing)

> Hermes Agent Skill — 上海嘉定融媒体中心记者秦建的新闻写作风格指南

## 简介

本 Skill 为 Hermes Agent 提供秦建（上海嘉定融媒体中心记者）的新闻写作风格参考，用于：

- 按秦建的写作风格撰写时政会议、民生服务、社会治理、文化历史、公共安全、产业经济等7类报道
- 覆盖四大平台：上海嘉定APP图文、电视新闻稿、抖音短视频脚本、嘉定报
- 包含完整的工作流（问材料→判断预告/现场→判断采访→按平台写稿→自检）
- 参考24位嘉定融媒体同侪记者的写作文本特征
- 基于838篇秦建工作文档 + 92篇平台稿件分析

## 核心能力

### 写作工作流（5步法）

1. **问材料** — 有没有素材？目标平台？截止时间？预告还是现场？
2. **处理材料** — 判断预告/现场，提取关键信息，官话转新闻语言
3. **判断采访** — 7种类型中5种需要采访引语
4. **按平台写** — APP图文/电视新闻/抖音脚本/嘉定报
5. **执行自检** — 11项质量检查

### 7种报道类型

| 类型 | 核心特征 |
|------|---------|
| 时政会议 | 领导讲话提炼 + 市民关联段 |
| 民生服务 | 实地探访 + 市民声音 + 数据支撑 |
| 社会治理 | 问题背景 + 行动过程 + 成效展示 |
| 文化历史 | 场景描写 + 嘉定特色 + 古今对照 |
| 公共安全 | 案件概要 + 警方提醒 + 安全防范 |
| 产业经济 | 项目展示 + 产业背景 + 政府服务 |
| 人物通讯 | 人物画像 + 故事叙述 + 当事人原话 |

### 4大平台

- **上海嘉定APP**：500-1200字，最完整形态
- **电视新闻**：240-360字，画面+口播
- **抖音短视频**：15-60秒，钩子式开头
- **嘉定报**：正式书面体，深度解读

## 触发方式

安装后在 Hermes 对话中使用以下关键词即可自动调用本 Skill：

### 直接触发

提到"写稿""写新闻""写报道"等词，Hermes 会自动加载 Skill 并按工作流执行：

```
帮我写一篇关于嘉定端午活动的新闻稿
写个嘉定APP的民生报道
这个会议帮我写个电视新闻稿
```

### 按平台指定

| 关键词 | 触发平台 |
|--------|----------|
| `APP` / `图文` / `嘉定APP` | 上海嘉定APP图文报道 |
| `电视` / `口播` / `电视新闻` | 电视新闻稿 |
| `抖音` / `短视频` / `脚本` | 抖音短视频脚本 |
| `嘉定报` / `报纸` | 嘉定报 |

### 按类型指定

| 关键词 | 触发报道类型 |
|--------|------------|
| `会议` / `调研` / `讲话` | 时政会议报道 |
| `民生` / `菜篮子` / `便民` | 民生服务报道 |
| `社区` / `消防` / `排查` | 社会治理报道 |
| `文化` / `节庆` / `非遗` | 文化历史报道 |
| `案件` / `反诈` / `安全` | 公共安全报道 |
| `企业` / `招商` / `产业` | 产业经济报道 |
| `人物` / `专访` / `故事` | 人物通讯报道 |

### 示例

```
写篇嘉定区防汛工作会议的APP报道        → 时政会议 + APP
寒潮来了，写个菜篮子供应的抖音脚本      → 民生服务 + 抖音
帮我把这份活动方案改成现场新闻稿         → 自动判断预告→现场
```

> 💡 如果提供了活动方案但活动时间在未来，Skill 会自动追问是写"预告"还是按"已举办"写现场报道。
